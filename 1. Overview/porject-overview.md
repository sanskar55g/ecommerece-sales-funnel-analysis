Project Overview

This project presents a comprehensive analysis of an ecommerce sales funnel using SQL as the core analytical engine. The primary objective is to
examine how users navigate through the buying journey, from initial platform discovery all the way to final purchase completion, and to quantify
the effectiveness of each stage in driving conversions.

Ecommerce businesses often struggle with identifying exactly where and why potential customers abandon their journey. This project addresses that
challenge by breaking the funnel into clearly defined stages including product discovery, product listing views, product detail page views, add 
to cart events, checkout initiation, and order completion. Each stage is analyzed independently as well as in relation to the stages before and 
after it, giving a holistic view of the entire conversion pipeline.

Using SQL, the raw transactional and behavioral data is queried, cleaned, and transformed to produce meaningful metrics. These include stage 
wise conversion rates, cumulative drop off rates, average time spent between stages, revenue leakage at each funnel step, and cohort level 
comparisons to understand how different user segments perform across the funnel.

The insights derived from the SQL analysis are then brought to life through an interactive Power BI dashboard. The dashboard is designed to give
business stakeholders a clear and intuitive visual understanding of funnel performance, highlighting critical drop off points, underperforming
stages, and areas with the highest potential for revenue growth. The combination of rigorous SQL analysis and dynamic Power BI visualizations 
makes this project a end to end solution for ecommerce funnel intelligence and data driven decision making.
